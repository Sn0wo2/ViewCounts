# ViewCounts

> ViewCounts API是由Golang开发的一个访客API计数

---

## 介绍:

可以设置一个IP计数一次，或一个IP访问X次计数一次，或一个IP在一段时间内访问只能记录一次且支持记录访问日志日志

---
